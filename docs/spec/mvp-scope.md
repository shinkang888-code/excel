# MVP 범위 및 지원 정책

## 목표
MVP는 사용자가 업로드한 `xlsx/xlsm` 문서를 분석하고, `Google 로그인` 후 `Google Sheets`로 새 파일을 생성하는 것을 목표로 한다. 핵심 가치는 `서식 최대 보존`, `실패 항목 가시화`, `결과 링크 즉시 제공`이다.

## 지원 범위 정의
서비스는 모든 엑셀 기능을 100% 동일하게 재현하는 제품이 아니라, `자동 변환 가능한 범위`와 `수동 보정이 필요한 범위`를 명확히 나누는 제품으로 정의한다.

### 1. 완전 지원 목표
아래 항목은 MVP에서 기본 지원 대상으로 본다.

- 워크북/시트 생성
- 셀 값
- 기본 수식
- 셀 병합
- 행 높이 / 열 너비
- 폰트, 글자색, 배경색
- 테두리, 정렬, 줄바꿈
- 숫자/날짜 표시 형식의 주요 패턴
- 숨김 시트 여부

### 2. 부분 지원 목표
아래 항목은 자동 변환을 시도하되, 결과가 일부 달라질 수 있다.

- 조건부 서식
- 데이터 유효성 검사
- 이미지 삽입 및 위치
- 차트 메타데이터 재구성
- 시트 보호 설정
- 필터 및 고정 행/열
- 복합 수식과 시트 간 참조

### 3. 미지원 또는 리포트 우선
아래 항목은 MVP에서 `자동 완전 변환`을 약속하지 않는다.

- VBA 매크로
- ActiveX / OLE 객체
- 외부 데이터 연결
- 피벗 테이블의 완전 동일 재생성
- 피벗 차트 자동 재생성
- 고급 조건부 서식 규칙 전부
- Excel 전용 함수 중 Google Sheets 비호환 함수

## 지원 정책 문구
아래 문구를 제품 전반의 표준 안내 문구로 사용한다.

### 업로드 전 안내
`기본 서식, 병합 셀, 주요 수식은 자동 변환됩니다. 차트, 피벗, 매크로, 일부 조건부 서식은 부분 지원 또는 수동 보정이 필요할 수 있습니다.`

### 변환 시작 전 안내
`이 변환은 Google Sheets의 표현 한계 내에서 최대한 원본을 보존합니다. 100% 픽셀 단위 동일성은 보장하지 않습니다.`

### 변환 완료 후 안내
`자동 보존된 항목과 수동 확인이 필요한 항목을 호환성 리포트에서 확인하세요.`

### 매크로 안내
`VBA 매크로는 Google Sheets에서 직접 실행되지 않습니다. 매크로는 감지 후 리포트로 안내되며, 추후 Apps Script 전환 보조 기능으로 확장할 수 있습니다.`

## 업로드 제한
초기 운영 안정성을 위해 아래 제한을 둔다.

- 파일 형식: `xlsx`, `xlsm`
- 파일 크기: 최대 `20MB`
- 시트 수: 최대 `50`
- 총 셀 수: 최대 `200,000`
- 동시 변환: 사용자당 최대 `3건`

## 성공 기준
MVP 성공 기준은 아래와 같이 둔다.

- 사용자가 업로드 후 `1분 이내`에 결과 링크 또는 실패 리포트를 받는다.
- 기본 서식 재현 만족도 `80% 이상`
- 완전 실패율 `5% 이하`
- `partial_success` 포함 전체 처리 성공률 `90% 이상`

## 상태 정의
- `uploaded`: 파일 저장 완료
- `analyzing`: 구조 분석 중
- `ready_for_export`: 미리보기 및 권한 요청 가능
- `authorizing_google`: Google Drive/Sheets 권한 요청 중
- `converting`: Google Sheets 생성 및 데이터 반영 중
- `partial_success`: 결과 생성 완료, 일부 항목 보정 필요
- `success`: 결과 생성 완료, 치명적 손실 없음
- `failed`: 결과 생성 실패

## 제품 메시지 원칙
- 지원하지 않는 기능을 숨기지 않는다.
- `완전 지원`, `부분 지원`, `미지원`을 업로드 직후부터 보여준다.
- 실패는 한 문장으로 끝내지 않고, 사용자 행동 지침을 반드시 제공한다.
- 고급 객체는 `자동 재생성 여부`보다 `감지 정확도`를 먼저 확보한다.
